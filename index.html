Projects

Tonealyzer

Ruby on Rails, React.js, Redux, PostgreSQL, D3.js

Tonealyzer is a full-stack web application where users can get tone analysis for writings they submit. They can also see how their writing tone has changed over time. The IBM Watson Tone Analyzer API provides the tone information for users' writing, from overall tone of their submissions, to specific analysis for each sentence.

OmniAuth was used to manage user login and security. Users can log in with Google, Facebook, LinkedIn, or basic username and password.

The D3.js library was used to create graphs where users can easily see the analysis of their writings over time.

Sharebnb

Ruby on Rails, React.js, Redux, PostgreSQL

Sharebnb is a full-stack web application inspired by Airbnb. Bookable home locations are displayed based on user search queries. A map is displayed using the Google Maps API, which users can adjust to specify their search.

Users are able to search for listings, as well as create, update, and delete their own listings. These listings will be available for other users to book.

Pacxon

JavaScript, EaselJS, HTML5, CSS3

Pacxon is an interactive game inspired by Xonix and Pacman, written in JavaScript. Pacman's goal is to move around the board, trapping the ghosts.

Vector math is used to determine the movement of and interaction between characters. The game also utilizes flood fill algorithm techniques to recursively identify which zones must be filled in, so that users experience interactive and dynamic game play.
